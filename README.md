# Java-Swing-Aplikacija (Client-Server)

Obicna Gui Aplikacija koja racuna linearne jednacine za 3 nepoznate.
Projekat demonstrira rad sa grafičkim korisničkim interfejsom (Swing) i mrežnu komunikaciju putem Socket programiranja.

## Funkcionalnosti

* `Sistem 3x3:` Rešavanje sistema linearnih jednačina sa tri nepoznate varijable.
* `Client-Server Arhitektura:` Klijent šalje koeficijente jednačine serveru, koji vrši matematički proračun i vraća rezultat.
* `Swing GUI:` Intuitivan korisnički interfejs za unos koeficijenata i prikaz rešenja u realnom vremenu.
* `Multithreading:` Server je dizajniran da podrži komunikaciju sa klijentom bez blokiranja glavne niti aplikacije.

## Tehnologije

* `Jezik:` Java
* `GUI Biblioteka:` Java Swing (AWT)
* `Mreža:` Java Sockets (TCP/IP)
